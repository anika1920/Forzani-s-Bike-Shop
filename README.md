📊 Bike Shop Sales & Performance Dashboard (Power BI)
📌 Project Overview

This project presents an interactive Power BI dashboard designed to analyze the business performance of a bike shop from 2023 to 2025.

The dashboard focuses on delivering clear insights into sales trends, purchasing behavior, cash flow, and product performance, helping stakeholders make data-driven decisions.

🎯 Objectives
Analyze yearly sales and purchase trends
Monitor cash inflow and outflow
Compare supplier and warehouse performance
Track units sold vs purchased
Understand category-wise contribution (Bicycles, e-Bikes, Parts)
📊 Dashboard Features
🔹 Key KPIs

📊 Category Contribution Analysis
📉 Cash Flow Waterfall Analysis
💡 Key Insights
Sales show a consistent upward trend from 2023 to 2025
Cash flow indicates significant fluctuations, especially in 2025
Inventory purchases increased earlier, but revenue realization lagged
Certain suppliers and warehouses outperform others consistently
